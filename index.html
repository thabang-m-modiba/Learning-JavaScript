Variables

In JavaScript, variables can be declared using var, let, or const.

Different ways to declare a variable.

  • Using var: var name = "Thabang";
  • Using let: let age = 25;
  • Using const: const IS_STUDENT = true;

Differences between var, let and const

  • var is function-scoped, while let and const are block-scoped.
  • var can be redeclared and updated, while let can be updated but not redeclared, and const cannot be updated or redeclared.
  • var is hoisted to the top of its scope and initialized with undefined, while let and const are hoisted but not initialized.

Variable names are case sensetive.

Naming Variables.

  • Variable names can contain letters, digits, underscores, and dollar signs.
  • Variable names must begin with a letter, underscore, or dollar sign.
  • Variable names cannot be JavaScript reserved words.
  • Variable names are case sensitive (y and Y are different variables).
  • Variable names should be meaningful.

The difference between declaring and Assigning variable.

Declaring a variable means creating it, while assigning a variable means giving it a value. We use the assignment operator = to assign a variable.

Increment and Decrement

Incrementing a variable means increasing its value by 1, while decrementing a variable means decreasing its value by 1.

  • Increment operator: ++
  • Decrement operator: --

Escape character

The backslash (\) is used as an escape character in JavaScript. It allows you to include special characters in a string.

  • \' - Single quote
  • \" - Double quote
  • \\ - Backslash
  • \n - New line
  • \r - Carriage return
  • \t - Tab
  • \b - Backspace
  • \f - Form feed

Concatenation

Concatenation is the process of joining two or more strings together. In JavaScript, we use the + operator to concatenate strings.

Example: "Hello, " + "world!" results in "Hello, world!"

We can also use the += operator to concatenate a string to an existing variable.

Example: let message = "Hello, "; message += "world!"; results in "Hello, world!"

Arithmetic operators

  • + (Addition)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)
  • % (Modulus)
  • ++ (Increment)
  • -- (Decrement)

Strings and dot length function.

Strings are used to represent text in JavaScript. They are enclosed in single or double quotes.

The length property of a string returns the number of characters in the string.

Example: "Hello".length results in 5

Immutability of Strings

In JavaScript, strings are immutable, which means that once a string is created, it cannot be changed. Any operation that seems to modify a string actually creates a new string.

Finding a character in a string

You can access individual characters in a string using bracket notation and the index of the character you want to access. Note that string indices are zero-based, meaning the first character is at index 0.

Example: "Hello"[0] results in "H"

Arrays

Arrays are used to store multiple values in a single variable. They are created using square brackets [].

Example: let fruits = ["Apple", "Banana", "Orange"];

Arrays can contain values of different data types, including strings, numbers, booleans, objects, and even other arrays.

You can access individual elements in an array using bracket notation and the index of the element you want to access. Note that array indices are zero-based, meaning the first element is at index 0.

Example: fruits[0] results in "Apple"

The length property of an array returns the number of elements in the array.

Example: fruits.length results in 3

Nested Arrays

Arrays can contain other arrays as elements, creating a nested structure. You can access elements in nested arrays using multiple indices.

Example: let matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];

Accessing an element in a nested array: matrix[1][2] results in 6

Modifying Arrays

You can modify elements in an array by accessing them via their index and assigning a new value. Example: fruits[1] = "Mango"; changes the second element from "Banana" to "Mango".

push()

The push() method adds one or more elements to the end of an array and returns the new length of the array.

Example: fruits.push("Grapes"); adds "Grapes" to the end of the fruits array.

pop()

The pop() method removes the last element from an array and returns that element. This method changes the length of the array.

Example: let lastFruit = fruits.pop(); removes "Grapes" from the end of the fruits array and stores it in the lastFruit variable.

shift()

The shift() method removes the first element from an array and returns that element. This method changes the length of the array.

Example: let firstFruit = fruits.shift(); removes "Apple" from the beginning of the fruits array and stores it in the firstFruit variable.

unshift()

The unshift() method adds one or more elements to the beginning of an array and returns the new length of the array.

Example: fruits.unshift("Strawberry"); adds "Strawberry" to the beginning of the fruits array.

Creating reusable code (Functions).

A function is a block of code designed to perform a particular task. It is executed when "something" invokes it (calls it).

Functions are defined using the function keyword, followed by the function name, parentheses (), and curly braces {}.

Function Parameters and Arguments

Functions can take parameters, which are placeholders for values that are passed to the function when it is called. The values passed to the function are called arguments.

Example: function greet(name) { return "Hello, " + name + "!";}

Return Statement

The return statement is used to specify the value that a function should return when it is called. If a function does not have a return statement, it will return undefined by default.

Example: function add(a, b) { return a + b; }

Variable scopes

Variable scope refers to the accessibility of variables in different parts of your code. In JavaScript, there are three types of scope: global scope, local scope, and block scope.

  • Global Scope: Variables declared outside of any function or block have global scope and can be accessed from anywhere in the code.
  • Local Scope: Variables declared within a function have local scope and can only be accessed within that function.
  • Block Scope: Variables declared with let or const within a block (e.g., inside an if statement or loop) have block scope and can only be accessed within that block.

Global vs Local Scope

Global variables can be accessed from any function, while local variables can only be accessed within the function they are declared in. It is generally a good practice to minimize the use of global variables to avoid potential conflicts and unintended side effects.

JSON.stringify

JSON.stringify() is a method that converts a JavaScript object or value to a JSON string. It is often used when sending data to a web server or storing data in a file.

Example: let obj = { name: "Thabang", age: 25 };
let jsonString = JSON.stringify(obj); results in '{"name":"Thabang","age":25}'

Boolean values

Boolean values represent one of two values: true or false. They are often used in conditional statements to control the flow of a program.

If statements

If statements are used to execute a block of code if a specified condition is true. They can also include else and else if clauses to handle multiple conditions.

Comparison operators

  • == (Equal to)
  • === (Strict equal to)
  • != (Not equal to)
  • !== (Strict not equal to)
  • < (Less than)
  • <= (Less than or equal to)
  • > (Greater than)
  • >= (Greater than or equal to)

Logical operators

  • && (Logical AND)
  • || (Logical OR)
  • ! (Logical NOT)

Difference between equality operator and strictly equality operator

The equality operator (==) compares two values for equality, but it performs type coercion if the values are of different types. The strictly equality operator (===) compares both the value and the type, and does not perform type coercion.

Example: 5 == "5" results in true, while 5 === "5" results in false

Switch statements

Switch statements are used to perform different actions based on different conditions. They are often used as an alternative to multiple if-else statements when you have a single variable to evaluate against multiple possible values.

Objects

Objects are used to store collections of data and more complex entities. They are created using curly braces {} and consist of key-value pairs.

Example: let person = { name: "Thabang", age: 25, isStudent: true };

You can access object properties using dot notation or bracket notation.

Accessing Object Properties

Example using dot notation: person.name results in "Thabang"

Example using bracket notation: person["age"] results in 25

Modifying Object Properties

You can modify object properties by accessing them via their key and assigning a new value. Example: person.age = 26; changes the age property from 25 to 26.

Adding New Properties

You can add new properties to an object by simply assigning a value to a new key. Example: person.address = "South Africa"; adds a new property address to the person object.

Deleting properties

You can delete properties from an object using the delete operator. Example: delete person.isStudent; removes the isStudent property from the person object.

Nested Objects

Objects can contain other objects as properties, creating a nested structure. You can access properties in nested objects using multiple keys.

Example: let student = { name: "Thabang", details: { age: 25, address: "South Africa" } };

Accessing a property in a nested object: student.details.age results in 25

Loops

Loops are used to execute a block of code multiple times. There are several types of loops in JavaScript, including for loops, while loops, and do-while loops.

  • For Loop: Used to iterate over a block of code a specific number of times.
  • While Loop: Used to iterate over a block of code as long as a specified condition is true.
  • Do-While Loop: Similar to the while loop, but the block of code is executed at least once before checking the condition.

Do-While loops

The do-while loop is similar to the while loop, but it guarantees that the code block will be executed at least once before the condition is tested. The syntax is as follows:

do {// code block to be executed} while (condition);

Nested Loops

Loops can be nested inside other loops to create more complex iterations. Example: A for loop inside another for loop to iterate over a 2D array.

Maths.random()

The Math.random() function returns a floating-point, pseudo-random number in the range 0 (inclusive) to 1 (exclusive). It is often used to generate random numbers for various purposes, such as games or simulations.

Example: let randomNum = Math.random(); results in a random number between 0 and 1.

To generate a random number within a specific range, you can use the following formula:

Math.floor(Math.random() * (max - min + 1)) + min;

This will generate a random integer between the specified min and max values (inclusive).

ParseInt()

The parseInt() function is used to convert a string into an integer. It takes two arguments: the string to be converted and an optional radix (base) that specifies the numeral system to be used.

Example: let num = parseInt("42"); results in 42

If the string cannot be converted to a valid integer, parseInt() will return NaN (Not a Number).

Ternary Operator

The ternary operator is a shorthand way of writing an if-else statement. It takes three operands: a condition, a value to return if the condition is true, and a value to return if the condition is false. The syntax is as follows:

condition ? valueIfTrue : valueIfFalse;

Example: let age = 18; let canVote = (age >= 18) ? "Yes" : "No"; results in "Yes"

Undefined vs Null

Undefined and null are both used to represent the absence of a value, but they have different meanings. Undefined means that a variable has been declared but has not been assigned a value, while null is an assignment value that represents no value or no object.

Example of undefined: let x; console.log(x); results in undefined

Example of null: let y = null; console.log(y); results in null

Object.freeze

The Object.freeze() method is used to freeze an object, preventing new properties from being added to it, existing properties from being removed, and existing properties (or their enumerability, configurability, or writability) from being changed. In essence, it makes the object immutable.

Example: let obj = { name: "Thabang" }; Object.freeze(obj); obj.name = "John"; console.log(obj.name); results in "Thabang"

Note that Object.freeze() only applies to the immediate properties of the object. If the object contains nested objects, those nested objects can still be modified unless they are also frozen.

Anonymous functions

An anonymous function is a function that does not have a name. It is often used as an argument to other functions or as an immediately invoked function expression (IIFE).

Example of an anonymous function as an argument: setTimeout(function() { console.log("Hello, World!"); }, 1000);

Example of an IIFE: (function() { console.log("Hello, World!"); })();

Arrow functions

Arrow functions are a shorthand way of writing anonymous functions. They use the => syntax and have a more concise syntax than traditional function expressions.

Example of an arrow function: let add = (a, b) => a + b;

Arrow functions do not have their own this value, which can be useful in certain situations, such as when working with callbacks or event handlers.

Calling Arrow funcions

Example of calling an arrow function: console.log(add(2, 3)); results in 5

console.log

The console.log() function is used to print messages to the console. It is often used for debugging purposes.

Destructing Assignment

Destructuring assignment is a syntax that allows you to unpack values from arrays or properties from objects into distinct variables. It provides a more concise and readable way to extract data.

Example of array destructuring: let [a, b] = [1, 2]; results in a = 1 and b = 2

Example of object destructuring: let {name, age} = {name: "Thabang", age: 25}; results in name = "Thabang" and age = 25

Template Literals

Template literals are a way to embed expressions within string literals, allowing for easier string interpolation and multi-line strings. They are enclosed in backticks (`) instead of single or double quotes.

Example of a template literal: let name = "Thabang"; let message = `Hello, ${name}!`; results in "Hello, Thabang!"

Template literals can also span multiple lines without the need for escape characters.

Example of a multi-line template literal: let multiLine = `This is line 1. This is line 2.`;

Simple Fields

Simple fields are a way to define properties on an object using a shorthand syntax. They allow you to create an object with properties that have the same name as the variables being assigned to them.

Example of simple fields: let name = "Thabang"; let age = 25; let person = { name, age }; results in { name: "Thabang", age: 25 }

Class

A class is a blueprint for creating objects with shared properties and methods. It is a way to define a custom data type in JavaScript.

Example of a class: class Person { constructor(name, age) { this.name = name; this.age = age; } greet() { return `Hello, my name is ${this.name} and I am ${this.age} years old.`; } }

Creating an instance of a class: let person1 = new Person("Thabang", 25);

Calling a method on an instance: console.log(person1.greet()); results in "Hello, my name is Thabang and I am 25 years old."

Constructor

The constructor is a special method that is called when a new instance of a class is created. It is used to initialize the properties of the object.

Example of a constructor: constructor(name, age) { this.name = name; this.age = age; }

Getters and Setters

Getters and setters are special methods that allow you to define how to access and modify the properties of an object. Getters are used to retrieve the value of a property, while setters are used to set the value of a property.

Example of a getter: let person = { firstName: "Thabang", lastName: "Mamoloko", get fullName() { return `${this.firstName} ${this.lastName}`; } }; Accessing the fullName property: console.log(person.fullName); results in "Thabang Mamoloko"

Example of a setter: let person = { firstName: "Thabang", lastName: "Mamoloko", set fullName(name) { [this.firstName, this.lastName] = name.split(" "); } }; Setting the fullName property: person.fullName = "John Doe"; results in person.firstName = "John" and person.lastName = "Doe"

Classes, Getters and Setters

Example of a class with getters and setters: class Person { constructor(firstName, lastName) { this.firstName = firstName; this.lastName = lastName; } get fullName() { return `${this.firstName} ${this.lastName}`; } set fullName(name) { [this.firstName, this.lastName] = name.split(" "); } }

Creating an instance of the class: let person1 = new Person("Thabang", "Mamoloko");

Accessing the fullName property: console.log(person1.fullName); results in "Thabang Mamoloko"

Setting the fullName property: person1.fullName = "John Doe"; results in person1.firstName = "John" and person1.lastName = "Doe"

Import and Export

In JavaScript, the import and export statements are used to share code between different files or modules. The export statement is used to make a variable, function, or class available for use in other files, while the import statement is used to bring in code from other files.

Example of exporting a function: export function greet(name) { return `Hello, ${name}!`; }

Example of importing a function: import { greet } from './greet.js'; console.log(greet("Thabang")); results in "Hello, Thabang!"
