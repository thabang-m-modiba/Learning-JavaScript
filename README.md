## Variables
In JavaScript, variables can be declared using ```var```, ```let``` or ```const```.

### Different ways to declare a variable.
* Using ```var```: ```var name = "Thabang";```
* Using ```let```: ```let age = 22;```
* Using ```const```: ```const IS_STUDENT = true;```
